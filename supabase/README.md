# Supabase — note operative

## RLS Policies
Le policy RLS sono versionate in `rls_policies.sql`.
Per applicarle su un nuovo progetto: aprire Supabase SQL Editor e eseguire il file.
Per aggiornare le policy: modificare il file E applicare la modifica su Supabase dashboard,
tenendo i due allineati.

## Backup
Eseguire un backup del DB prima di qualsiasi modifica a policy RLS o schema.
